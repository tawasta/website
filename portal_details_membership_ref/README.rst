=============================
Portal Details Membership Ref
=============================
* Portal details (/my/account/) shows Membership Reference (partner.ref)

Configuration
=============
* None needed

Usage
=====
* Fill in the "Membership Number" for a partner in backend
* Log in to that partner's My Account view and see the field in the sidebar
