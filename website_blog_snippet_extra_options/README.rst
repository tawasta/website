==========================================
Website 'Blog Posts' Snippet Extra Options
==========================================

* Adds extra configurations to the Blog Posts snippet in frontend
* New filters:

  * Promoted latest blog posts
  * Promoted most viewed blog posts

Configuration
=============
* In backend event form, mark the blog as promoted
* Configure the Blog Posts snippet itself via website builder
